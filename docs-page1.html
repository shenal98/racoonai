Model and Tool Gallery


Tools to analyze data:
Analyze My Data

This toolkit provides a rich insight in to your dataset. Our functions evaluate your dataset on-the-spot and provides analysis helping you to identify impurities and biases in your data set.
Principal Component

Principal Component Analysis, or PCA, is a dimensionality-reduction method that is often used to reduce the dimensionality of large data sets, by transforming a large set of variables into a smaller one that still contains most of the information in the large set.
Preprocess Data

This tool takes your dataset in to our system for preprocessing and data cleaning. You will be able to download your dataset after the cleaning process. The cleaned dataset then can use through our machine learning model gallery.
ML Models:
Linear Regression

This machine learning algorithem allows you to identify linear pattern between two variables in your dataset. After feeding enough data, you will be able to provide a reasonable prediction based on our model.
Multiple Regression

This machine learning algorithem allows you to identify linear pattern between two or more variables in your dataset. After feeding enough data, you will be able to provide a reasonable prediction based on our model.
Decision Tree

Classification is a two-step process, learning step and prediction step, in machine learning. In the learning step, the model is developed based on given training data. In the prediction step, the model is used to predict the response for given data. Decision Tree is one of the easiest and popular classification algorithms to understand and interpret.
Artificial Neural Network

This tool allows you to build and test a simple artificial neural network and use it to find hidden relationships in your data.After training your ANN model, you will be able to predict defined outcomes to given inputs.
